This project develops computational methods for analysing and modelling musical style. Using source-separated audio recordings and machine learning techniques, we investigate how rhythmic, harmonic, and timbral features combine to create distinctive performer identities. Our work on jazz piano style has demonstrated that machine learning models can reliably identify performers from their rhythmic patterns alone.
