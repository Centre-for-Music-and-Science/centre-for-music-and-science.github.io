This project investigates how jazz musicians achieve coordination in real-time ensemble performance. Using computational analysis of timing patterns in source-separated audio recordings, we study the trade-offs musicians make between rhythmic precision and expressive flexibility, including how network delay affects coordination in online performance settings.
